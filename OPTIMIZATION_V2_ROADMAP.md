## 已实现功能

### ✅ literary_metrics.py — 文学质感量化引擎

**纯规则统计，零 LLM 成本**，从原文直接计算：

| 指标 | 说明 | 效果 |
|------|------|------|
| 模板词密度 | 6类35个标记词频次/万字 | 套路化检测 |
| 对话占比 | 中文引号内文字比例 | 叙事方式判断 |
| 感叹号/省略号密度 | 标点使用频率 | 风格判断 |
| 平均句长 + 长句率 | 句子长度统计 | 文笔特征 |
| 字符多样性 | 去重字符/总字符 | 信息密度 |
| 章节长度变异系数 | std/mean × 100 | 节奏均匀度 |
| 程序化评分 | 文笔/节奏/情绪各0-10 | 交叉验证用 |

实测区分度：
- 凡人修仙之仙界篇：模板词23.8/万，文笔4.8分
- 大唐贞观第一纨绔：模板词9.5/万，文笔5.5分
- 大奉打更人：模板词6.0/万，文笔6.3分

### ✅ 速读卡片

报告最前面的一键概览：
- 综合推荐等级（S/A/B/C/D）
- 预估阅读时间（万字/小时）
- 类型标签
- 一句话评价
- 评分条（■■■■■□□□□□）+ 可信度标记（🟢🟡🔴）
- 高潮片段 / 低谷区域
- 风险 / 亮点

### ✅ 评分可信度报告

LLM radar_scores 与规则统计交叉验证：
- 🟢 高可信：LLM 与规则评分差 ≤ 1.5
- 🟡 中可信：偏差 1.5-3.0 或仅有一路信号
- 🔴 低可信：仅一路评分或严重矛盾

### ✅ 模拟读者书评

基于报告数据自动生成口语化书评（不调 LLM）：
- 综合推荐 + 类型 + 概括
- 亮点和风险
- 节奏/情绪/文笔特征
- 适合读者群体
- 最强/最弱维度

### ✅ 关键场景概要

从 chunk_results 提取 one_sentence_summary，按冲突/伏笔/爽点重要度排序展示 top 15。

### ✅ 节奏叙事

基于 chunk 密度标签生成口语化节奏描述：
- 全书节奏偏快/紧凑/均衡/偏慢
- 节奏切换频率
- 最紧凑/最平淡处

### ✅ 设定细节

从 chunk worldbuilding 字段提取具体设定条目（去重后最多12条）。

### ✅ 避雷提示（详细）

从 chunk quality_notes 和 summary risks 中提取具体风险标签：
- 匹配虐心/背叛/后宫倾向/战力崩坏/水文/烂尾等关键词
- 标注具体内容出处

## 三、V3 优化方案：感性量化为数学计算

### 3.1 核心思路

"写作和阅读是感性的" → 可以转化为可计算的数学指标：

| 感性体验 | 量化方法 | 数学基础 | 可信度标注 |
|---------|---------|---------|-----------|
| "文笔好不好" | 模板词密度 + 字符多样性 + 句长方差 | 频率/方差/信息熵 | 采样覆盖率 |
| "节奏好不好" | 章节长度均匀度 + 张力曲线斜率 + 节奏切换频率 | 标准差/一阶差分 | 有效章节比 |
| "情绪被调动" | 情感词分布 + 正负情绪比率 + 情绪波动幅度 | 词频/极性/振幅 | 词典匹配率 |
| "读起来顺不顺" | 对话/叙述比 + 段落长度分布 + 连接词密度 | 香农熵 + 统计分布 | 采样覆盖 |
| "有没有深度" | 信息密度 + 伏笔回收率 + 语义层丰富度 | 信息论 + 图论 | LLM交叉验证 |
| "值不值得看" | 加权综合评分 + 可信度标注 + 同类对比 | 贝叶斯估计 + 置信区间 | 多路交叉 |

### 3.2 设计原则

1. **不纯算法解决感性问题** — 规则统计提供基线，LLM 提供语义理解，两者交叉验证
2. **量化有边界** — 每个指标标注可信度和样本量，不装全知
3. **零额外 LLM 成本** — 新增分析全部基于纯规则统计
4. **渐进式集成** — 有开关控制，可单独关闭

- 内置约 800 个中文情感词（正面 400+ / 负面 400+）
